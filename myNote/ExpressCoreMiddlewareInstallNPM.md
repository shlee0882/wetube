
### Express Core 미들웨어 여러가지 설치

### morgan
```bash
npm install morgan
```
- logging에 도움을 준다.

### helmet
```bash
npm install helmet
```
- nodejs 앱 보안을 제공한다. 

### cookie-parser
```bash
npm install cookie-parser
```
- cookie 다루는 것을 제공한다.

### body-parser
```bash
npm install body-parser
```
- body로 부터 정보를 받게 한다.
